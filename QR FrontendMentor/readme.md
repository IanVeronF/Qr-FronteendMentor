# Componente Código QR

Este proyecto es una práctica de Frontend Mentor, donde se construye un **componente de tarjeta con código QR** usando **HTML, CSS y JS básico**. El objetivo es replicar el diseño lo más fiel posible y practicar buenas prácticas de desarrollo web.

## Vista previa

![Vista previa del componente](images/image-qr-code.png)

## Cómo abrir el proyecto

1. Clona este repositorio o descarga los archivos.
2. Abre `index.html` en tu navegador.
3. ¡Listo! Verás la tarjeta con el código QR centrada en la pantalla.

## Tecnologías utilizadas

- HTML5
- CSS3 (Flexbox, variables CSS)
- Fuente: [Outfit](https://fonts.google.com/specimen/Outfit)
- Opcional: JavaScript básico para futuras interacciones
